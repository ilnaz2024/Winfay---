# 🌐 Winfay — На шаг впереди

Добро пожаловать в репозиторий **Winfay**! Это всемирная, независимая и безопасная десктопная социальная сеть, написанная на Python с использованием графического интерфейса PyQt6 и базы данных SQLite3.

## ✨ Основные фичи:
* 👑 **Иммунитет Создателя**: Встроенная защита аккаунта главного разработчика (`coolboy337`).
* 🔄 **Авто-вход**: Умная система запоминает сессию и пускает в ленту без повторного ввода пароля.
* 👍👎 **Интерактивная лента**: Система лайков и дизлайков в реальном времени.
* 🗑️ **Модерация**: Возможность полного удаления постов.

## 🚀 Как запустить:
1. Установите Python и библиотеку PyQt6 (`pip install PyQt6`).
2. Запустите файл командой: `python app.py`
